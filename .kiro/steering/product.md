# Product Overview

EduHelper is an educational platform that connects teachers and students in a digital learning environment.

## Core Features

- Role-based access (Teacher/Student portals)
- Teacher capabilities: manage classes, students, assignments, and track progress
- Student capabilities: access courses, complete assignments, view grades, and track learning progress
- Dashboard-based interface with real-time activity tracking
- Assignment management and grading system

## User Roles

- **Teachers**: Create lessons, manage students, track progress, grade assignments
- **Students**: Access courses, complete assignments, view grades, manage schedule

## Current State

The application is in active development with basic dashboard interfaces implemented. Backend API structure is in place with routes for authentication, teachers, students, classes, and assignments.
